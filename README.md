# Language app to learn Finnish

This is the first complete "app" for language learning, instead of just single features loosely connected to one another. Currently focused on learning Finnish in English.

The story feature is by far the most important one. It has audio, and synchronized, advanced subtitles. The first row of subtitles are subword-level, like pöydä-llä -> table-on. The second row is spoken Finnish, and the third row is a normal English translation.
